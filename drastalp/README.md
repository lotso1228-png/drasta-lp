# ドラスタ自動車教習所 — ランディングページ

LINE予約でラクラク教習。最短ルートで普通AT免許取得をサポートする **ドラスタ自動車教習所** の
ランディングページです。Claude Design のモックアップをもとに、依存ゼロの静的サイト
（HTML / CSS / Vanilla JS）として実装しています。GitHub + Vercel でそのままデプロイできます。

## 構成

```
index.html      # ページ本体（セマンティックHTML）
styles.css      # スタイル（CSS Grid / Flexbox・レスポンシブ）
main.js         # ヘッダー縮小・スクロール表示・FAQ開閉
assets/         # 使用画像（透過PNG素材・写真・favicon）
vercel.json     # Vercel 設定（静的配信・キャッシュ）
project/        # デザイン元データ（モックアップ・全素材・引き継ぎ資料）
chats/          # デザイン制作時の会話ログ
```

ビルド不要です。ローカル確認は任意の静的サーバーで：

```bash
python3 -m http.server 8000   # → http://localhost:8000
```

## セクション

ヘッダー（スクロールで縮小） / ヒーロー / 選ばれる理由 / コース・料金 /
免許取得までの流れ / 通いやすい環境 / 試験にかかる費用 / よくある質問 /
LINE相談CTA / フッター

インタラクション：ボタンホバー、スクロールでの各セクションのフェードイン、
ナビの下線アニメーション、FAQアコーディオン（同時に1つだけ開く）。
レスポンシブ対応（PC / タブレット / スマホ）。

## 画像について（重要）

元のモックアップにあった一部の写真（ヒーロー、流れの各ステップ、運転席）は
素材パックに実写真が含まれていなかったため、**同梱の無料素材イラスト・写真を仮画像**として
配置しています。実写真に差し替える場合は、`assets/` に画像を置き、`index.html` の該当 `<img src>`
を差し替えるだけです。差し替え箇所は以下のとおり：

| 場所 | 現在の仮画像 | 差し替え推奨 |
|------|------------|------------|
| ヒーロー | `asset_084.png` | 車内のカップル / 受講生の写真 |
| 流れ ①〜⑥ | `asset_076 / 074 / 081 / 085 / 091 / 082` | 各ステップの実写真 |
| 環境「1コマ50分」 | `asset_090.png` | 運転席・コックピットの写真 |

## デプロイ（GitHub → Vercel）

1. **GitHub にプッシュ**

   ```bash
   git remote add origin https://github.com/<あなたのユーザー名>/<リポジトリ名>.git
   git push -u origin main
   ```

2. **Vercel にインポート**
   - [vercel.com/new](https://vercel.com/new) で上記リポジトリを選択
   - Framework Preset は **Other**（静的サイト）／Build Command・Output Directory は空のまま
   - **Deploy** を押すと公開されます

`index.html` がリポジトリのルートにあるため、追加設定なしで配信されます。
